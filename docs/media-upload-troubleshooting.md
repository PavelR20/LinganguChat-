# Media Upload Troubleshooting Guide

If you are unable to send photos, videos, or other files through the LinganguChat application, follow the steps below to diagnose and resolve the problem. Completing each section increases the chances of restoring normal media sharing functionality without reinstalling the entire system.

## 1. Confirm the Basics
- **Check your internet connection.** Try loading a web page or another streaming app. Media uploads require a stable connection; if Wi-Fi is unreliable, switch to mobile data (or vice versa).
- **Verify LinganguChat is up to date.** Visit the Play Store/App Store and ensure you have the latest version. Updates often patch upload failures.
- **Restart the device.** Power cycling clears temporary glitches that can block file transfers.

## 2. Validate Permissions
- Open the device settings and locate LinganguChat in the application list.
- Make sure storage, files, and camera permissions are enabled.
- If any permission is disabled, turn it on and retry sending an image from the chat.

## 3. Inspect Storage Availability
- Confirm that your device has at least 500 MB of free storage. Low space can prevent new media from being processed.
- Remove unused files or clear cache if storage is below this threshold.

## 4. Clear Cache (Android)
1. Go to **Settings → Apps → LinganguChat → Storage**.
2. Tap **Clear cache** (avoid clearing data unless cache alone fails).
3. Reopen LinganguChat and attempt to send the image again.

## 5. Reset Network Settings (Optional)
- For persistent network-related failures, reset your device's network settings.
- Note that this will remove saved Wi-Fi passwords and Bluetooth pairings, so back them up first.

## 6. Test on Another Network or Device
- Try sending the same image using a different Wi-Fi network or mobile data.
- If possible, log into LinganguChat on a second device. Success there indicates the issue is specific to the original device configuration.

## 7. Capture Logs for Support
- Navigate to **Settings → About → Build number** within the app (if available) and follow on-screen instructions to generate a support log.
- Share the generated log with the development team for further diagnostics.

## 8. Contact Support
If none of the above steps resolves the issue:
- Email **support@linganguchat.example.com** with details including device model, OS version, app version, and any error messages.
- Attach screenshots or screen recordings of the failure if possible.

Following this checklist should resolve most scenarios where media fails to send in LinganguChat. These steps mirror the triage flow used by the support team, ensuring consistency between user troubleshooting and internal diagnostics.
